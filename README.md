# Бесконечное ЛНМО

Обычный парень по имени Михаил, выпустившись из ЛНМО, поступил на МКН, но вылелел оттуда на 3-м курсе. И вот ему уже 30 лет, он работает инженером и ведёт спокойную жизнь. У него есть жена и двое детей.

Одним судьбоносным вечером он, закончив отчёт по работе, натыкается в ящике на распечатку своей старой научной работы. Воспоминания о былых днях наполняют его, и он переносится в летнюю школу ЛНМО...
